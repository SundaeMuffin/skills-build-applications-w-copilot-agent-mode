## What is GitHub Copilot Cloud Agent?

GitHub Copilot Cloud Agent (formerly *Copilot coding agent*) is an AI-powered autonomous software engineer that lives directly inside GitHub. Unlike the in-editor Copilot assistant, Cloud Agent can:

- **Read and understand issues** — it is assigned to a GitHub issue just like a human developer.
- **Plan and execute multi-step tasks** — it reasons about what needs to be done and carries it out step by step.
- **Write, edit, and commit code** — it makes real changes to files in the repository and pushes them to a pull request.
- **Run tests and validate changes** — it can execute the repository's existing test suite to verify its own work.
- **Respond to review feedback** — comments left on the pull request are read and acted upon automatically.
